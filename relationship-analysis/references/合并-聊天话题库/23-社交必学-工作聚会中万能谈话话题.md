# 工作聚会中万能谈话话题

> **核心**：找共同点——共同的兴趣、经历、共同点。

---

## 和领导聊

| 问题 | 效果 |
|------|------|
| "我看到你去了……" | 展示你在关注他，让他感到被重视 |
| "你是怎么进入这个行业的？" | 人都喜欢聊自己的经历 |
| "你今年有什么目标？" | 了解公司重点，顺势展示你也关心这些 |
| "职业生涯早期得到的最好的建议是什么？" | 请教姿态，屡试不爽 |

---

## 和所有人聊

| 问题 | 效果 |
|------|------|
| "这份工作让你最喜欢的是什么？" | 和工作有关，又足够私人，容易找到共鸣 |
| "最近看了什么电影/书？" | 延伸话题无穷无尽 |
| "今年去度假了吗？去了哪？" | 人人都喜欢聊假期，无论他去了还是待家里 |

---

## 心态提醒

开放式问题比封闭式问题好——能引发讨论的问题比"是/否"回答更有价值。
