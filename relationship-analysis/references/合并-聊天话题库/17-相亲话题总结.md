# 相亲话题总结

> **首选话题**：先聊兄弟姐妹，比聊父母更安全（万一父母离异会引起不快），也更容易赢得信任。

---

## 话题清单

| 话题 | 适用场景 | 说明 |
|------|----------|------|
| 兄弟姐妹 | 首选 | 了解家庭背景，活跃气氛 |
| 儿时的理想 | 轻松愉快 | 童年回忆容易拉近距离 |
| 喜欢的美食 | 餐厅/酒吧 | 天然契合，可延伸到下次邀约 |
| 名胜古迹 | 旅行爱好者 | 让她介绍，你做听众 |
| 业余爱好 | 了解生活方式 | 运动、音乐、绘画等 |
| 大学生活 | 怀旧 | 走出社会的人都怀念 |
| 工作情况 | 了解现状 | 同行业找共鸣，不同行业长见识 |
| 身边的好友 | 了解社交圈 | 看他看重朋友的什么特质 |
| 想去的旅游地 | 延伸话题多 | 可自然引出下次一起旅行 |
| 喜欢的歌/电影/书 | 了解品味 | 也是潜在共同兴趣 |
| 感情经历 | 敏感 | 视关系深度酌情使用 |
| 今天趣事/心情 | 日常 | 坦诚分享最容易 |

---

## 补充话题

- **时事政治**：看对方兴趣，政审话题不要强聊
- **体育新闻**：男生通常感兴趣
- **适当赞赏优点**：真诚的赞美能拉近距离，但不要过度

---

## 心态提醒

- 第一次见面以了解为主，不要一上来就查户口
- 先听她说，再适时分享自己
- 她说得多的地方就是兴趣点，顺着挖
