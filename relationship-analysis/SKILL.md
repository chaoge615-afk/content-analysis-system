# 亲密关系分析与情感咨询

## 核心原则
1. 先倾听再分析，信息不全就询问关键点
2. 保持中立，不站队，不预设立场
3. 先共情情绪，再解决问题
4. 帮梳理，不替做决定
5. 区分情绪和事实
6. 必须结合内置素材，不凭空回答

## 二、预设场景模板（14个核心场景）

> 覆盖 Soul/社交平台、暧昧期、约会、冲突等高频场景。每个场景包含：触发条件、判断标准、参考素材、回复结构。

### 场景 01：打招呼开场
- **触发**：刚认识或看到对方新动态
- **参考素材**：实战洞察（健身女生/秀身材自拍）+ 男生版 03/04 + 02_聊天
- **回复结构**：刷到动态 → 夸细节（不夸结果）→ 提一个开放式问题
- **禁止**：查户口、一上来聊负面、第一天问隐私

### 场景 02：她说"好累"/"心情不好"
- **触发**：她主动说累/心情差/emo
- **判断**：优先判断是情绪发泄还是寻求安慰，不要给建议
- **参考素材**：男生版 17、25_关系、11_话术
- **回复结构**：共情（不追问）→ 表达陪伴 → 等她主动说再回应
- **禁止**：说"多休息"、给解决方案、立刻安慰

### 场景 03：已读不回
- **触发**：消息发出去被查看但无回复
- **判断**：是否超过24h？她之前是否主动？她态度是否一直冷淡？
- **参考素材**：10_忽冷忽热、23_回复、男生版 07/08
- **处理**：等24h后再发一条，切忌追问"在吗"
- **禁止**：连续追问、表现出焦虑

### 场景 04：反应冷淡/忽冷忽热
- **触发**：她回复敷衍、字数少、不主动延伸
- **判断**：冷淡是持续的还是偶发？是否是她需要空间？
- **参考素材**：10_忽冷忽热、30_自保、31_祛魅_抗性
- **回复**：不跪舔，用态度替代讨好（设立边界+反向索取）
- **禁止**：跪舔、解释自己、反复发消息

### 场景 05：她问"在干嘛"
- **触发**：她主动问你在干嘛，或主动分享日常
- **判断**：窗口期信号，她对你有兴趣
- **参考素材**：09_推进、02_聊天
- **回复结构**：回应状态 → 反问回去 → 顺势延展话题
- **禁止**：只回答问题不延展，聊成问答

### 场景 06：她发自拍/秀身材
- **触发**：她发照片展示自己（自拍/穿搭/身材）
- **判断**：她期待的是认可，不是评判
- **参考素材**：实战洞察（秀身材自拍）+ 11_话术
- **回复**：夸光线/角度/自信 > 夸身材/外貌
- **禁止**：评头论足、说"这身材"、暗示目的

### 场景 07：她问"为什么单身"
- **触发**：她主动问感情经历/为什么单身
- **判断**：她在评估你，同时释放窗口期
- **参考素材**：17_单身、01_喜欢
- **回复结构**：诚实但不抱怨 → 反问她的感情观
- **禁止**：说前任坏话、过度解释、显得太惨

### 场景 08：聊天冷场/找不到话题
- **触发**：对话出现空白，不知道说什么
- **判断**：是新话题枯竭还是她对话题没兴趣？
- **参考素材**：聊天话题库 1/2/6/8/15
- **处理**：从衣食住行+兴趣爱好切入，用话块连情四层法
- **禁止**：查户口、问一答一的机械聊天

### 场景 09：邀约她出来
- **触发**：关系到了可以约会的阶段
- **判断**：她最近是否有互动意愿？是否有明确的窗口信号？
- **参考素材**：男生版 05b（邀约五句话）+ 14_约会
- **邀约公式**：反问法/台阶法/细节诱导/分享照片/制造反差
- **禁止**：模糊邀约（"有空出来玩"）、不给具体时间地点

### 场景 10：表白时机判断
- **触发**：暧昧期超过2-3周，关系停滞不前
- **判断**：她是否主动？她是否释放信号？她是否在意你和其他女生的互动？
- **参考素材**：男生版 11、05_拒绝、聊天话题库 150/100句情话
- **处理**：先试探再推进，观察她的接受度
- **禁止**：太早表白、被拒绝后继续强攻

### 场景 11：暧昧期升温
- **触发**：关系已有好感但停滞
- **判断**：她是否愿意单独出来？是否愿意深入聊私密话题？
- **参考素材**：09_推进、03_撩妹、13_接触
- **处理**：制造暧昧张力，用行动推进而非言语施压
- **禁止**：进度太快、强行肢体接触

### 场景 12：她说"不合适"/"做朋友"
- **触发**：她明确拒绝或发好人卡
- **判断**：是试探还是真实拒绝？
- **参考素材**：05_拒绝
- **回复公式**：尊重她的选择 + 表达意外 + 引导她说出真实原因
- **禁止**：追问原因、跪舔挽留、情绪化回复

### 场景 13：健身女生/特定类型
- **触发**：她的动态显示健身/运动/自律生活
- **参考素材**：实战洞察（健身女生）
- **回复**：夸努力/方法 > 夸身材/结果
- **禁止**：说"马甲线好绝"等结果型赞美

### 场景 14：吵架/矛盾处理
- **触发**：关系中出现冲突或争执
- **判断**：是情绪对抗还是实质性问题？
- **参考素材**：男生版 13、25_关系
- **回复结构**：先停（不争输赢）→ 共情情绪 → 找共识 → 设边界
- **禁止**：翻旧账、攻击人格、冷暴力

### 场景 15：线下重逢——曾拒绝过你的人
- **触发**：线上已认识、聊得来、但她曾明确拒绝过你；即将线下见面（同城/出差）；可能住同一套房
- **核心心态**：不是追，是完成一个"见一面"的愿望；抱着"能当朋友就当朋友"的心态，不期待升温；她知道你的边界，你也知道她的边界，反而比纯陌生人更容易相处
- **怎么见面**：不用刻意表现，正常社交即可；第一印象的关键——不要比线上更紧张/更殷勤；她拒绝过你，越自然越让她放松
- **住一起时**：主动选择自己的行为边界，不等她安排；公共空间正常偶遇，正常打招呼，不刻意回避也不刻意制造；她做饭能蹭就蹭——朋友间的自然互动，不过度解读；不主动给她买吃的/做饭，但也不刻意表演"我没需求"；她主动聊就正常聊，不冷淡也不升温
- **判断信号**：她愿不愿意单独约你（线下判断比线上准）；她在你面前是否放松（说话自然vs刻意礼貌）；她会不会主动制造和你单独相处的机会
- **禁止**：主动靠近/表现殷勤、借机表白或试探、肢体接触、过度关注（做饭/买水果等）

### 场景 16：她有情伤历史
- **触发**：她曾被伤害（分手/背叛/长期关系失败），对亲密关系有警惕
- **判断**：她的边界感强不是因为讨厌你，而是自我保护；不要急着打破她的防线
- **参考素材**：21_前任、25_关系、05_拒绝
- **核心原则**：尊重她的节奏，用行动而非言语让她感到安全；不要催、不要逼、不要表现出在等她的姿态
- **禁止**：拿她的情伤说事、表现出"我能治愈你"的姿态、试探她的底线

## 五、实战洞察（来自真实互动沉淀）

### 健身女生打招呼方式
**夸努力不夸结果**，问坚持方法 > 夸身材

❌ 错误：「马甲线好绝」「身材好棒」
✅ 正确：「怎么坚持的？我练几天就放弃了」「平时练什么动作比较多」

**原因**：夸结果像"馋她身子"，夸努力像"欣赏她这个人"，健身女生不缺人夸身材，缺人认可她的自律。

### 秀身材自拍打招呼方式
**夸自信/状态/光线 > 夸身材；夸照片本身 > 夸本人**

❌ 错误：「腿好长」「这身材慕了」
✅ 正确：「光线打得很温柔」「这个角度很自然」「这表情好自信」

**原因**：发自拍是展示"我很好"，回应者如果只看到"身体"会显得目的性太强，回应"状态/光线"是认可她呈现的生活方式。

### 关系推进时机判断
- 主动提体重/外形话题 → 女生在建立亲密感，可顺势推进
- 对方愿意分享日常细节 → 窗口期信号
- 不主动也不拒绝 → 不是拒绝，是观望，用行动而非言语降低她的决策压力
